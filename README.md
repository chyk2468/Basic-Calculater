# My Super Cool Calculator

Hey there! Thanks for checking out my project. I'm just starting out with web development, and this is my very first calculator. It's not NASA-level tech, but it actually works!

## What can it do?
- Add stuff: 1 + 1 = 2 (Mind-blowing, right?)
- Subtract stuff: For when you have 10 cookies and eat 5.
- Multiply and Divide: For when math gets scary.
- Clear button: If you mess up (happens to the best of us).
- Decimal points: Because whole numbers are boring.

## How I built it
- HTML: The bones of the app. I used a table because it's easier to align buttons.
- CSS: Added some colors and shadows so it doesn't look like it's from 1995.
- JavaScript: The "brain." I used something called eval() to do the math. (I heard it's a bit "dangerous" for big apps, but for this, it's perfect!)

## How to run it
1. Download these files.
2. Double-click index.html.
3. Profit!

## A little note
I'm still learning, so if the code looks a bit messy, that's why! Feel free to play around with it or change the colors.
